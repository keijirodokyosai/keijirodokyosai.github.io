================================
■更新手順
================================

1 GitHubでファイルを編集
2 Commit
3 GitHub Pages自動更新
4 数分後に反映


================================
■トラブル対応
================================

【画像が表示されない】

・imagesフォルダに配置されているか確認
・ファイル名のスペル確認


【PDFがダウンロードできない】

・pdfフォルダに配置されているか確認
・リンクURL確認


【GitHub Pagesが更新されない】

・Commitが完了しているか確認
・数分待つ
・ブラウザキャッシュを更新
